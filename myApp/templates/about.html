About Kordle

Inspiration

Kordle is inspired by LoLdle and Wordle, a game where you have to guess the champion or a word. This game is built using the Django REST Framework for the backend, while the frontend is made with HTML, CSS, and JavaScript.

How to Play

Enter a idol's name in the text box and click Guess to submit your answer. The game provides visual feedback through color-coded tiles:

Green: The category is correct.
Red: The category is incorrect.
↑ (up arrow): The answer is higher then the guess.
↓ (down arrow): The answer is lower then the guess.

If all the tiles are green, you guessed the correct idol! An example is shown below:

Use your Kpop knowledge to guess the idol!

Scoring

You earn 1 point for each correctly guessed idol.

Reloading the page resets the idol to a new random selection.
